## [ESEM16简介](https://github.com/SoCXin/ESEM16/wiki)

[ESEM16](https://github.com/SoCXin/ESEM16) 32K+2K存储,两个16位定时器,一路UART,一个32位定时器,支持ISP，IAP，SWD功能。

#### 关键特性

* 差分输入信号范围±600mV
* 有功电能计量误差小于0.1%，动态范围为1000：1（@ 25℃）
* 一路电流和一路电压有效值测量误差小于1%，动态范围为500：1（@ 25℃）
* ADC：12位精度，10通道
* 3.0V ~ 3.6V
* 封装SSOP24
